# Setting Up and Running the Project

## 1. Add Your Asset

Place your image in the `images/` directory and ensure your `pubspec.yaml` includes:

```yaml
flutter:
  assets:
    - images/diamond.png
```

## 2. Get Packages

Run this in the terminal:

```bash
flutter pub get
```

## 3. Run the App

```bash
flutter run
```

Make sure a simulator or connected device is running.
